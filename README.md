## Usage

### Timer
1. Click the **MINUTNIK** (Timer) button
2. Set the desired minutes and seconds
3. Click **START** to begin countdown
4. Use **PAUZA** (Pause) to pause, **WZN√ìW** (Resume) to continue
5. Click **RESET** to reset to initial values

### Stopwatch
1. Click the **STOPER** (Stopwatch) button
2. Click **START** to begin counting
3. Use **PAUZA** (Pause) to pause, **WZN√ìW** (Resume) to continue
4. Click **RESET** to reset to 00:00:00

## Controls

| Button | Function |
|--------|----------|
| START | Start the timer/stopwatch |
| PAUZA | Pause the running timer/stopwatch |
| WZN√ìW | Resume after pause |
| RESET | Reset to initial state |
